# Lab pfSense – Segmentation réseau

## Objectif
Créer un réseau entreprise avec séparation entre employés et invités.

## Architecture
Internet → pfSense → LAN (Employés) + OPT1 (Invités)

## Technologies
- pfSense
- VirtualBox
- Ubuntu

## Résultat
- Réseau employés isolé
- Invités ont accès à internet uniquement
- Accès LAN bloqué pour invités
